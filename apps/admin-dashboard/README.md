# 🎛️ Admin Dashboard (The Control Tower)

**Part of the [OmniRoute Sentinel](../../README.md) Ecosystem.**

This is the central command center for fleet managers. It provides a stunning, high-performance interface to monitor, dispatch, and reroute vehicles in real-time.

## ✨ Key Features
- **Live Fleet Tracking:** Visualize all vehicles and their coordinates on a live map.
- **One-Click Dispatch:** Send new trucks to specific cities instantly.
- **Dynamic Override:** Intervene and reroute shipments based on live external threat intel.

## 🚀 Getting Started

This application is built with Next.js (App Router).

### Running Locally

If you're running the entire Turborepo ecosystem, this app starts automatically on `http://localhost:3000`.

To run just this app:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Tech Stack
- Next.js (App Router)
- React 19
- Tailwind CSS
- Zustand (State Management)
- Leaflet (Maps)
- Socket.io Client
